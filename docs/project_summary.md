# Project Summary: TCGA-BRCA Differential Expression Analysis

This mini-project analyses public TCGA-BRCA breast cancer RNA-seq data using R and Bioconductor.

## What the project does

The workflow compares gene expression between primary breast tumor samples and solid tissue normal samples. It identifies differentially expressed genes, visualises the results, explores survival differences based on ERBB2 expression, trains a LASSO classifier, and performs Gene Ontology enrichment analysis.

## Main methods

- TCGA data retrieval with TCGAbiolinks
- RNA-seq differential expression analysis with DESeq2
- Volcano plot visualisation with EnhancedVolcano
- Heatmap visualisation with pheatmap
- Kaplan-Meier survival analysis with survival/survminer
- LASSO logistic regression with glmnet
- Gene Ontology enrichment with clusterProfiler
